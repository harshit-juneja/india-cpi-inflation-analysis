# india-cpi-inflation-analysis

## objective
Analyze India's CPI inflation trends across different categories to identify key drivers and economic patterns.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization
- Data Cleaning

##  Dataset
CPI inflation data across multiple categories such as Food, Fuel, Housing, etc.

## Key Analysis Performed
- Trend analysis of inflation over time
- Category-wise inflation comparison
- Identification of peak inflation periods
- Comparative analysis across sectors

## Key Insights
- Food category showed highest volatility in inflation
- Certain periods experienced sharp inflation spikes
- Fuel prices had a strong impact on overall CPI trends

## Outcome
This project demonstrates the ability to analyze economic data, build dashboards, and derive meaningful insights for decision-making.
